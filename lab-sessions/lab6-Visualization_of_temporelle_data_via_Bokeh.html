Bokeh for Time Series Analysis

**Introduction to Bokeh** Bokeh is an interactive visualization library for Python that targets modern web browsers for presentation. Unlike Matplotlib, which is primarily designed for static plots, Bokeh excels at creating interactive plots and dashboards. It can handle large datasets and streaming data, making it suitable for real-time applications.

Key Features of Bokeh:

  • Interactivity: Built-in support for zooming, panning, hovering, and other interactive tools.
  • Web-Focused: Generates HTML and JavaScript, making it easy to embed plots in web pages.
  • High Performance: Can handle large datasets efficiently.
  • Versatility: Supports a wide range of plot types (lines, bars, scatter plots, etc.).

Documentation:

For comprehensive documentation, please refer to the official Bokeh website: https://docs.bokeh.org/en/latest/

from statsmodels.tsa.stattools import acf
import numpy as np 

# Nombre d'observations
N = len(df["#Passengers"])

# ACF simple
acf_values = acf(df["#Passengers"], nlags=30)
lags = np.arange(len(acf_values))

# Seuil de confiance 95% : 1.96 / N
conf_interval = 1.96 / np.sqrt(N)
conf_low = [-conf_interval] * len(lags)
conf_high = [conf_interval] * len(lags)

# DataFrame pour Bokeh
acf_data = pd.DataFrame({
    'lag': lags,
    'acf': acf_values,
    'conf_low': conf_low,
    'conf_high': conf_high
})
source = ColumnDataSource(acf_data)

# Figure
p = figure(title="Autocorrelation Function (ACF)",
           x_axis_label="Lag", y_axis_label="Autocorrelation",
           width=900, height=400,
           tools="pan,wheel_zoom,box_zoom,reset,save")

# Barres
p.segment(x0='lag', y0=0, x1='lag', y1='acf', source=source, line_width=2, color="steelblue")
p.circle(x='lag', y='acf', source=source, size=5, color="steelblue")

# Bande de confiance 1.96/sqrt(N)
band = Band(base='lag', lower='conf_low', upper='conf_high', source=source,
            level='underlay', fill_alpha=0.2, fill_color='lightblue')
p.add_layout(band)

# Ligne horizontale y=0
zero_line = Span(location=0, dimension='width', line_color='black', line_width=1)
p.add_layout(zero_line)

show(p)

# Affiche image si le graphique ne s'affiche pas
img = mpimg.imread('images/graph/acf.png')
imgplot = plt.imshow(img)
plt.show()
