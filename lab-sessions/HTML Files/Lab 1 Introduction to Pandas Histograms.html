**Slectionner des colonnes d'un DataFrame. **

Pour slectionner des colonnes spcifiques d'un jeu de donnes (DataFrame) Pandas, deux mthodes principales sont disponibles : l'oprateur point (.) et l'oprateur crochets ([]).

Accs simple :

Par exemple, pour accder la colonne nomme cut du DataFrame diamonds_df, vous pouvez utiliser :

diamonds_df.cut

# ou 
diamonds_df['cut']

Cas o l'oprateur point (.) ne peut pas tre utilis :

L'oprateur point est pratique, mais il prsente des limitations dans certains cas :

Noms de colonnes contenant des espaces : Si un nom de colonne inclut des espaces (par exemple, 'prix au carat'), vous devez imprativement utiliser la notation avec les crochets :

diamonds_df['prix au carat']

Noms de colonnes entiers : Lorsque le nom d'une colonne est un nombre entier (par exemple, 2023), l'accs via l'oprateur point n'est pas valide. Utilisez les crochets :

diamonds_df[2023]

Cration de nouvelles colonnes : Pour ajouter une nouvelle colonne au DataFrame, la notation avec les crochets est la mthode standard :

diamonds_df['nouvelle_colonne'] = ...
diamonds_df['une autre colonne'] = ...
