Introduction aux Diagrammes Barres

Les diagrammes barres sont un outil de visualisation fondamental utilis pour afficher des donnes catgorielles avec des barres rectangulaires. La longueur de chaque barre est proportionnelle la valeur qu'elle reprsente. Les diagrammes barres sont excellents pour comparer des quantits entre diffrentes catgories.

Quand utiliser les diagrammes barres :

  • Comparaison de donnes catgorielles.
  • Affichage de la frquence ou du nombre de diffrentes catgories.
  • Visualisation des tendances entre les catgories.

Terminologie de base :

  • Barres : Les colonnes rectangulaires reprsentant les valeurs des donnes.
  • Axes : Les lignes horizontales (axe des x) et verticales (axe des y) qui dfinissent le graphique.
  • tiquettes : Le texte qui dcrit les catgories et les valeurs.
  • Titre : La description gnrale du graphique.

import pandas as pd
import matplotlib.pyplot as plt

# Compter le nombre de diamants par catgorie de 'cut'
cut_counts = diamonds_df['cut'].value_counts()

# Crer le diagramme  barres
plt.bar(cut_counts.index, cut_counts.values)

# Ajouter un titre et des tiquettes aux axes
plt.title("Nombre de diamants par catgorie de 'cut'")
plt.xlabel("Catgorie de 'cut'")
plt.ylabel("Nombre de diamants")

# Afficher le graphique
plt.show()

import pandas as pd

# Compter le nombre de diamants par catgorie de 'cut'
cut_counts = diamonds_df['cut'].value_counts().reset_index()
cut_counts.columns = ['cut', 'count']

# Crer un diagramme  barres interactif avec Plotly
fig = px.bar(
    cut_counts,
    x='cut',
    y='count',
    title="Nombre de diamants par catgorie de 'cut'",
    labels={'cut': "Catgorie de 'cut'", 'count': "Nombre de diamants"},
)

fig.show()
